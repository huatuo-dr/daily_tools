# 日常工具集开发里程碑 📋

## 项目概述
构建一个基于React的日常工具集网页，提供开发工具、天气、日历和新闻功能，采用左侧竖向Tab布局。

## 技术栈

- **框架**: React 18+
- **构建工具**: Vite 6.0+
- **样式方案**: 原生CSS（模块化）
- **状态管理**: React Hooks（useState, useEffect）
- **农历库**: lunar-javascript
- **图表绘制**: 原生SVG
- **代码规范**: ESLint 9+

---

## 开发规范

- 组件采用函数式组件 + Hooks
- 代码注释使用英文
- Git提交信息遵循Conventional Commits
- 代码需通过ESLint检查
