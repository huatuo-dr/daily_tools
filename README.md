# 日常工具集 🛠️

一个基于React构建的现代化日常工具集Web应用，集成开发工具、天气预报、日历黄历和新闻聚合功能。

[![React](https://img.shields.io/badge/React-18.3.1-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-6.0.1-646CFF.svg)](https://vitejs.dev/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**工具数量**: 5个主工具 + 6个开发子工具 + 5个摸鱼游戏 = 16个实用工具

## ✨ 功能特性

### 💻 Coding工具（新增）

#### 🔢 计算器
- **标准模式**：基础四则运算
- **程序员模式**：
  - 四种进制转换（HEX/DEC/OCT/BIN）实时显示，带空格格式化
  - 六种位运算（AND/OR/XOR/NOT/<</>>）
  - 十六进制字母输入（A-F）
  - 横向布局，一屏完整显示
  - 智能禁用：BIN模式禁用2-9，OCT模式禁用8-9

#### 📝 字符串处理
- **大小写转换**：大写、小写、首字母大写
- **字符串操作**：反转、去空格、删除所有空格
- **编码解码**：Base64、URL编码/解码
- **HTML转义**：防止XSS攻击
- **统计功能**：字符数、单词数、行数统计
- **行号显示**：便于查看多行文本
- **一键复制**：处理结果快速复制

#### { } JSON工具
- **实时校验**：边输入边检查JSON格式
- **格式化**：自动美化JSON结构
- **压缩**：移除空格和换行
- **详细错误**：精确显示错误位置和原因
- **示例加载**：快速加载示例JSON
- **行号显示**：便于定位错误位置
- **一键复制**：复制格式化结果

#### 🔍 正则测试
- **正则编辑器**：支持5种标志（g/i/m/s/u）
- **实时匹配**：输入即测试，立即显示结果
- **高亮显示**：黄色背景标记所有匹配项
- **详细信息**：显示匹配位置、捕获组内容
- **10个示例**：邮箱、手机、URL、身份证、日期、IP等
- **替换功能**：支持捕获组引用（$1, $2...）
- **一键复制**：快速复制替换结果
- **行号显示**：便于定位匹配位置

#### 🤖 命令翻译（AI驱动）
- **自然语言→命令**：将日常描述转为 Linux / PowerShell / CMD 命令
  - 例如："查找本目录下所有 *.log 文件"
  - 例如："列出当前目录的文件和大小"
- **命令→自然语言**：解释命令的含义和作用
  - 例如：`Get-Command python` → 解释其功能
  - 例如：`find . -name "*.txt" -mtime -7` → 解释其作用
- **AI流式响应**：实时显示翻译过程
- **聊天式界面**：支持多轮对话，优化命令
- **Markdown渲染**：代码块高亮显示
- **环境变量配置**：安全管理API密钥
- **Coze API集成**：纯前端实现，无需后端

#### 🌐 IP查询
- **IP归属地查询**：查询任意 IP 的地理位置信息
- **一键查本机**：快速获取当前网络出口 IP
- **详细信息**：国家、省份、城市、ISP、时区、经纬度
- **地图定位**：点击在高德地图中查看位置
- **常用示例**：Google DNS、Cloudflare、国内 DNS 等
- **免费 API**：使用 ip-api.com，无需配置 API Key

### 🌦️ 天气工具

- **真实数据**：集成和风天气 API，提供准确的天气信息
- **当前天气**：实时温度、天气状况、湿度、风力、降雨概率、温度范围
- **24小时预报**：逐小时温度曲线图和降雨概率曲线，1小时间隔，支持横向滚动
- **15天预报**：未来两周天气趋势，包含最高/最低温度、风力、降雨
- **城市搜索**：支持中文城市名搜索
- **智能降级**：API 调用失败时自动使用模拟数据

### 📅 日历工具

- **月视图日历**：清晰的月份日历展示，可点击日期
- **农历信息**：显示农历日期、节日、二十四节气（金色醒目标签）
- **节假日标注**：2024-2025年中国法定节假日高亮显示
- **调休标识**：标注调休上班日期
- **月份切换**：便捷的上月/下月/今天导航
- **黄历面板**：
  - 干支纪年（年月日天干地支）
  - 宜忌事项（绿红双栏显示）
  - 冲煞、五行、胎神信息
  - 吉神凶煞详情

### 📰 新闻工具

- **热点聚合**：来自TrendRadar的多平台新闻热点
- **自动加载**：页面打开自动获取最新内容
- **手动刷新**：一键更新新闻内容
- **时间显示**：显示上次刷新时间
- **完整展示**：保留原页面所有功能和样式

### 🐟 MOYU 摸鱼游戏

伪装成命令行终端的小游戏合集，摸鱼神器！

- **终端UI**：macOS 风格仿真终端，伪装工作状态
- **命令交互**：通过输入命令与游戏互动
- **历史记录**：上下键追溯历史命令
- **文本选择**：支持鼠标选择复制终端内容

#### 🎮 游戏列表（5个）

| 游戏 | 命令 | 说明 |
|------|------|------|
| **24点** | `24` | 给定4个数字（1-13），用 +-×÷ 计算出24，支持无解判断 |
| **猜数字** | `guess` | 猜4位不重复数字，系统反馈 XAYB（A=位置正确，B=数字正确） |
| **打字测试** | `type` | 英文句子打字练习，统计 WPM 和正确率 |
| **数字谜题** | `puzzle` | 找规律猜下一个数字，支持查看答案 |
| **算术挑战** | `math` | 限时口算挑战，支持难度和时间配置（默认60秒） |

#### 📋 通用命令

| 命令 | 说明 |
|------|------|
| `help` | 显示帮助信息 |
| `games` | 显示可用游戏列表 |
| `start N` / `N` | 启动指定游戏（N=序号或命令） |
| `exit` | 退出当前游戏 |
| `clear` | 清空屏幕 |

#### 🎯 游戏特色

- **24点**：`xxxx` 判断无解，`skip` 跳题，`score` 查看得分
- **猜数字**：`history` 查看猜测历史，`giveup` 放弃并查看答案
- **打字测试**：`new` 换题，`stats` 查看统计
- **数字谜题**：`hint` 提示，`answer` 查看答案，`skip` 跳题
- **算术挑战**：`time N` 设置时间（10-300秒），`restart` 重新开始，`easy/normal/hard` 切换难度

## 🚀 快速开始

### 环境要求

- Node.js 16.0+
- npm 7.0+ 或 pnpm

### 安装依赖

```bash
npm install
```

### 环境变量配置

部分功能需要配置 API Key。复制 `.env.example` 并创建 `.env.local` 文件：

```bash
cp .env.example .env.local
```

编辑 `.env.local`，填入你的配置：

```env
# 命令翻译工具（Coze API）
VITE_COZE_API_TOKEN=your_api_token_here
VITE_COZE_BOT_ID=your_bot_id_here
VITE_COZE_USER_ID=your_user_id_here
VITE_COZE_BASE_URL=https://api.coze.cn

# 天气工具（和风天气 API）
VITE_QWEATHER_API_KEY=your_qweather_api_key_here
```

> **注意**：`.env.local` 已在 `.gitignore` 中，不会提交到Git仓库。

#### 获取 API Key

- **和风天气**：[注册地址](https://dev.qweather.com/) | [配置指南](./doc/和风天气API配置指南.md)
- **Coze AI**：[注册地址](https://www.coze.cn/)

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动。

### 生产构建

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产构建

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 🔧 开发规范

### 代码风格

- 组件：使用函数式组件 + Hooks
- 命名：驼峰命名法（组件大驼峰，变量小驼峰）
- 注释：关键逻辑添加英文注释
- 格式：通过ESLint检查

### Git提交规范

遵循 Conventional Commits 规范：

```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 重构
perf: 性能优化
test: 测试相关
chore: 构建/工具链相关
```
